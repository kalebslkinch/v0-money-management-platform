# DWK SRD Baseline Expectations

## General System Requirements
* **[span_0](start_span)SRD-G01**: The system MUST provide secure login authentication, role identification, and two-factor authentication for all users.[span_0](end_span)
* **[span_1](start_span)SRD-G02**: The system MUST provide visible help and support access on all pages.[span_1](end_span)
* **[span_2](start_span)SRD-G03**: The system MUST maintain consistent interface structure, navigation, and interaction logic across all modules.[span_2](end_span)
* **[span_3](start_span)SRD-G04**: The system MUST clearly display data privacy, consent, and data usage information on key pages.[span_3](end_span)
* **[span_4](start_span)SRD-G05**: The system MUST support customizable data visualization with filtering, categorization, and flexible display options.[span_4](end_span)
* **[span_5](start_span)SRD-G06**: The system MUST provide a unified export function for reports and visualizations.[span_5](end_span)
* **[span_6](start_span)SRD-G07**: The system SHOULD provide global search and filtering functions across modules.[span_6](end_span)
* **[span_7](start_span)SRD-G08**: The system SHOULD maintain consistent branding, including typography, colours, and component styles.[span_7](end_span)
* **[span_8](start_span)SRD-G09**: The system SHOULD provide standardized report templates with limited customization options.[span_8](end_span)
* **[span_9](start_span)SRD-G10**: The system COULD support both manual logout and automatic logout after inactivity.[span_9](end_span)
* **[span_10](start_span)SRD-G11**: The system COULD provide onboarding guidance for first-time users.[span_10](end_span)
* **[span_11](start_span)SRD-G12**: The system COULD allow users to save customized dashboards and frequently used views.[span_11](end_span)

* **[span_13](start_span)SRD-U02**: The system MUST allow users to create custom transaction categories and tags.[span_13](end_span)

* **[span_17](start_span)SRD-U06**: The system MUST allow users to generate personalized financial reports and spending trend analysis.[span_17](end_span)
* **[span_18](start_span)SRD-U07**: The system MUST allow users to submit financial adviser consultation requests.[span_18](end_span)
* **[span_19](start_span)SRD-U08**: The system MUST allow users to share authorized financial summary data with advisers.[span_19](end_span)
